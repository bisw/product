
The Product Import module allows the admin to import content from the json file.

Installation
------------
Standard module installation applies.

Menus
-----
admin/config/content/product_import

Settings
--------
The settings page is at Administer >> Config >> Content >> Product Import.

Paste all the content from the json file and click import button to import all the content from the json file.

This is where you can import all the content.
